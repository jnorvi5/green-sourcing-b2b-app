## Color Coding Reference

### Status Colors
- **Green (Emerald)**: #10b981
  - Accepted quotes
  - Good response time (< 12 hours)
  - Positive trends

- **Yellow (Amber)**: #f59e0b
  - Pending quotes
  - Warning response time (12-24 hours)
  - Neutral states

- **Red**: #ef4444
  - Rejected quotes
  - Critical response time (> 24 hours)
  - Urgent deadlines

- **Teal (Primary)**: #14b8a6
  - Active/selected states
  - Primary actions
  - Brand color

- **Purple**: #a78bfa
  - Funnel chart
  - Special highlights

### Chart Colors
- **Win Rate**: Teal (#14b8a6)
- **Response Time Gauge**: Green/Yellow/Red (dynamic)
- **Revenue**: Emerald with gradient (#10b981)
- **Funnel**: Purple (#a78bfa)

## Responsive Breakpoints

| Screen | Width | Layout Changes |
|--------|-------|----------------|
| Mobile | < 640px | Single column cards, FAB visible, Bottom nav visible |
| Tablet | 640-1024px | 2-column cards, No FAB, No bottom nav |
| Desktop | > 1024px | 4-column cards, Full 2-column layout |

## Accessibility Features

### ARIA Labels
- `aria-label` on all icon-only buttons
- `aria-expanded` on dropdown triggers
- `aria-current="page"` on active nav items
- `aria-hidden="true"` on decorative icons
- `role="article"` on notification items
- `role="dialog"` on modal

### Keyboard Navigation
- Tab order follows visual hierarchy
- Enter/Space activates buttons
- Escape closes modals
- Arrow keys in dropdowns (future)

### Screen Reader Support
- Semantic HTML structure
- Descriptive button text
- Status announcements
- Form field labels

## Animation & Transitions

### Micro-interactions
- Button hover: Scale 1.05 (0.2s)
- Card hover: Background fade (0.2s)
- Modal open: Fade in (0.3s)
- Progress bars: Width transition (0.5s)
- Loading spinner: Continuous rotation

### Page Transitions
- Stats cards: Stagger fade-in
- Charts: Data animation (Recharts)
- Notifications: Slide down

## Z-Index Hierarchy

1. Modal backdrop: 50
2. Modal content: 50
3. Bottom nav: 50
4. FAB: 40
5. Dropdown: 50
6. Sticky headers: 10
7. Normal content: 1

This ensures proper layering without conflicts.
