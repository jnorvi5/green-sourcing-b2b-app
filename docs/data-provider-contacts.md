# GreenChainz Priority Data Provider Contacts
**Status**: Initial Research Phase  
**Last Updated**: November 10, 2025

## Priority 1: Core EPD & Sustainability Data

### 1. EPD International (The International EPD System)
- **Data Type**: Environmental Product Declarations (EPD), LCA data
- **Access**: Free API + Downstream REST API (ILCD/EPD JSON/XML format)
- **Contact**: https://www.environdec.com (look for "Developers" or "API Access")
- **Notes**: Core source for third-party verified life-cycle data. Free API access. Advisable to schedule intro call.
- **Action**: Draft partnership email requesting developer API access

### 2. WAP Sustainability
- **Data Type**: Scope 3 Emissions, Product Carbon Footprints (PCFs), OpenEPD Database, LCA aggregated data
- **Access**: Licensed REST API (OpenEPD Format)
- **Contact**: https://www.wapsustainability.com (contact form for "Startup Pricing")
- **Notes**: Hundreds of thousands of aggregated EPD/PCF records. Essential for carbon accounting. Requires annual licensing negotiation.
- **Action**: Inquiry email for startup pricing tier

### 3. Building Transparency
- **Data Type**: EPD database, certification data for construction materials
- **Access**: Free platform + API/export request
- **Contact**: https://www.buildingtransparency.org (look for "Data Access" or "Developers")
- **Notes**: Free but must request API access or bulk exports
- **Action**: Submit data access request via contact form

### 4. Autodesk Sustainability Data API
- **Data Type**: Standardized environmental data for building materials
- **Access**: Partnership email for developer access (Beta program)
- **Contact**: https://aps.autodesk.com/sustainability-data-api
- **Notes**: Critical if targeting AEC (architecture/engineering/construction). Currently in beta.
- **Action**: Email developer relations for partnership inquiry

### 5. FSC (Forest Stewardship Council)
- **Data Type**: Forestry certifications, supply chain data
- **Access**: Public database + API (documentation needed)
- **Contact**: https://fsc.org (search "Public Database" or "Developer API")
- **Notes**: API available. Need to review documentation for integration requirements.
- **Action**: Research API docs + email if commercial partnership needed
